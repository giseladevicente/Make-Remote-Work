# Make Remote Work - Tailwind CSS 4

Responsive landing page construida con **HTML, JavaScript y Tailwind CSS v4**.

Este proyecto recrea el desafío de **Frontend Mentor - "Sección introductoria con un sistema de navegación y menús desplegables"**, utilizando Tailwind CSS 4 para el estilo.

## Características

* Diseño responsive para **mobile**, **tablet** y **desktop**
* Menú de navegación con **dropdown**
* Menú **hamburguesa** para navegación en dispositivos móviles
* Animaciones interactivas en las flechas del menú
* Estilos desarrollados con **Tailwind CSS v4**
* Uso de **tipografía personalizada**

## Instalación y configuración

1. **Clonar el repositorio**

```bash
git clone https://github.com/giseladevicente/Make-Remote-Work.git
```

2. **Instalar dependencias**

```bash
npm install
```

3. **Generar los estilos de Tailwind**

 ```bash
npm run dev
```

4. Ejecutar el proyecto utilizando un servidor local.
